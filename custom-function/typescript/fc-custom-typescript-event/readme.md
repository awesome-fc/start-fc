# Alibaba Cloud Function Computing Custom Typescript Event function

You can quickly experience one-click deployment of a Typescript application on Alibaba Cloud Function Computing Service in just a few steps:

- Initialize the project: `s init fc-custom-typescript-event -d fc-custom-typescript-event`
- Enter the project: `cd fc-custom-typescript-event`
- Deployment project: `s deploy -y`
- Invoke function: `s invoke -e "hello"`
